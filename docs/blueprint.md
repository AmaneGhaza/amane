# **App Name**: Aman Gaza (Safe Gaza)

## Core Features:

- Multi-language Support: Full website translation in Arabic (default, RTL), French, and English, driven by JSON language files.
- User Registration & Verification: Secure registration process for beneficiaries and donors, with verification to ensure trust and legitimacy. The AI tool checks information entered by beneficiaries to automatically match them with appropriate categories of aid.
- Help Request Submission: Simplified form for Gazan users to submit help requests, categorized for medical, food, shelter, etc.
- Donor Matching: Matching algorithm that connects beneficiaries with appropriate donors based on location, needs, and urgency. This feature uses an AI tool that factors in donor preferences to create personalized aid packages for benficiaries. The algorithm is transparent and auditable to ensure legitimacy.
- Advanced Search: Smart search bar enabling users to find help by city, type of aid, urgency, or donor name.
- Transparent Statistics Display: Display of verified data showing donors, families matched, and help delivered to build trust.
- Contact & Support: Multiple contact channels including form, WhatsApp, and email, plus a map of Gaza.

## Style Guidelines:

- Primary color: Red (#B22222), derived from the Palestinian flag, signifying courage and resilience.
- Background color: Off-white (#F5F5F5), offering clarity and readability in a stressful context.
- Accent color: Green (#558B2F), another color from the flag, to foster a sense of hope and growth.
- Body and headline font: 'PT Sans' (humanist sans-serif) for readability and a modern, yet warm feel.
- Simple, visually clear icons to explain actions, avoiding any technical or political messaging.
- Lightweight, responsive layout designed for low-bandwidth conditions and readable on low-end phones. Large, high-contrast buttons and readable text are critical.
- Subtle animations such as fades and slides when content loads to give a smooth feel.
